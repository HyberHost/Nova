Nova WHMCS Integration Guide

Before continuing, please ensure you have gone through the files header.html and footer.html. These are where you need to add your own logo, update your links where necessary as well as the links to your social media pages.

Step-by-Step Integration

  1. Access WHMCS Admin Area
    Log in to your WHMCS admin area using your administrator credentials.
  2. Navigate to Email Settings
    In the WHMCS admin area, go to "Setup" and select "General Settings." Then, click on the "Mail" tab.
  3. Customize Email Templates
    Under the "Mail" tab, you'll find settings for customizing email templates. Follow these instructions to integrate your custom HTML and CSS email templates:
    • Header
      Open your header.html file and copy the HTML code from this file. In WHMCS, paste the HTML code into the "Email Header" section.
    • Footer
      Open your footer.html file and copy the HTML code from this file. In WHMCS, paste the HTML code into the "Email Footer" section.
    • Global CSS
      Open your global.css file and copy the CSS code from this file. In WHMCS, paste the CSS code into the "Global CSS" section.
  4. Save Changes
    After pasting the code into the respective sections, click the "Save Changes" button to save your email template customizations.
